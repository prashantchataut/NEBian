## Production Checklist

**Equipment:**
- [ ] Smartphone with good camera (iPhone or mid-range Android)
- [ ] Phone tripod or stable surface
- [ ] Natural light or ring light
- [ ] Screen recording software (OBS for desktop, built-in for mobile)

**Editing:**
- [ ] CapCut (free, powerful)
- [ ] Add captions/subtitles (algorithm loves this)
- [ ] Use chapters/timestamps for longer videos
- [ ] Add end screen with subscribe button + related video

**SEO:**
- [ ] Keyword-rich title ("NEB Grade 12 Physics", "SEE preparation")
- [ ] Detailed description with timestamps
- [ ] Tags: NEB, SEE, Nepal, board exams, study tips
- [ ] Thumbnail: Clear text, face, contrasting colors

**Posting Schedule:**
- Monday: Past paper solution (long-form)
- Wednesday: Study tip / strategy (medium)
- Friday: App feature / tutorial (short)

*YouTube is a long game. First 10 videos might get 50 views each. Video 50 might get 5,000. Consistency wins.*
